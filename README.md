# AWS-IAM-Security-Audit
This repository contains my Identity and Access Management(IAM) project completed during my internship.

Intern ID: CITS4223

Name: Smrithi S

Duration: 6 Weeks

Project Name: AWS IAM Security Audit

Project Scope: This project involves implementing and managing AWS Identity and Access Management (IAM) security controls within an AWS cloud environment. The implementation includes creating IAM users and groups, configuring custom and inline IAM policies, applying role-based access control, enabling multi-factor authentication (MFA), configuring password security policies, testing authorized and unauthorized access scenarios, setting up IAM Access Analyzer for identifying unintended access, and configuring AWS CloudTrail for monitoring and auditing user activities. The project demonstrates secure identity management, permission control, least privilege implementation, and security auditing practices within the AWS cloud environment.
